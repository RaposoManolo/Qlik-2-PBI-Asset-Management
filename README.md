**Asset Management Dashboard – Qlik to Power BI Migration**

This project recreates an asset management dashboard originally built in Qlik, focusing on asset performance, lifecycle, and key KPIs. It demonstrates cross-platform capability by translating Qlik Set Analysis into DAX, enabling flexible and scalable reporting. The project also leverages Power BI Project (PBIP) for Git-based version control and collaborative development, aligning with modern BI DevOps practices.

**Key Features**

Asset overview KPIs 
Trend analysis <br>
Data modelling approach <br>
Designed for stakeholder reporting <br>
